OSDR_readthedocs tutorial using the mkdocs basic template

Welcome to Open Science Data Repository (OSDR) tutorial documentation!

OSDR is a public science data library for space biology researchers and future space explorers. It pulls data from the https://osdr.nasa.gov/bio/ and offers a simple and intuitive API.

Check out the API usage section for further information, including how to installation the project.

Note

This project is under active development.
